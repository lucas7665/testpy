# 更新日志

## 2025-10-30 - 信号记录功能

### ✨ 新增功能

#### 📋 信号记录表格
- 显示最近的买入/卖出信号记录
- 实时更新，自动记录交易信号
- 只显示有效信号（买入/卖出），过滤观望状态

#### 表格字段
| 字段 | 说明 |
|------|------|
| 时间 | 信号发生时间（分钟级） |
| 价格 | 当时的螺纹钢价格 |
| RSI6 | 短线RSI指标值 |
| RSI12 | 中线RSI指标值 |
| RSI24 | 长线RSI指标值 |
| 信号 | 信号类型（买入/强烈买入/卖出/强烈卖出） |

#### 信号类型
- 🔔 **买入信号** - RSI6 < 30
- 🔔🔔 **强烈买入** - RSI6 < 30 且 RSI12 < 40（多周期共振）
- ⚠️ **卖出信号** - RSI6 > 70
- ⚠️⚠️ **强烈卖出** - RSI6 > 70 且 RSI12 > 60（多周期共振）

#### 视觉效果
- ✅ 价格上涨显示绿色
- ❌ 价格下跌显示红色
- 🎨 不同信号类型用不同颜色徽章标识
- 🖱️ 鼠标悬停高亮行

### 🎨 界面优化
- 图表高度固定为400px，查看更舒适
- X轴优化，最多显示10个刻度，避免拥挤
- Y轴显示格式优化

### 📊 数据展示
- 最多显示最近20条信号记录
- 自动过滤观望状态
- 实时更新，无需刷新页面

## 使用说明

### 如何查看信号记录

1. 启动Web服务器
2. 打开浏览器访问页面
3. 滚动到"📋 信号记录"部分
4. 查看历史买入/卖出信号

### 如何使用信号记录

1. **查看历史信号** - 了解最近的市场信号趋势
2. **验证策略** - 对比历史信号和实际价格走势
3. **把握时机** - 根据多周期共振信号做决策
4. **风险控制** - 观察信号频率和准确度

## 技术实现

- 前端：JavaScript动态生成表格
- 数据源：从API获取的历史RSI数据
- 更新频率：每10秒自动刷新
- 数据量：保留最近100条数据，显示最近20条信号
